# GROUP-NUMBER-4-CALENDER-APP
Android Calender App
PROJECT NAME: Android Calendar App

GROUP NUMBER:4

DESCRIPTION:
This project is a mobile calendar application developed using Kotlin in Android Studio. 
The app allows users to:

- Select dates using CalendarView
- Add events
- View saved events
- Receive notifications for scheduled events

TECHNOLOGIES USED:
- Kotlin
- Android Studio
- XML Layouts
- SharedPreferences / File Storage
